# 快速开始指南

## 5分钟快速上手

### 1️⃣ 安装依赖 (1分钟)

```bash
cd frontend
npm install
```

### 2️⃣ 启动开发服务器 (30秒)

```bash
npm run dev
```

打开浏览器访问: **http://localhost:3000**

### 3️⃣ 登录 (1分钟)

使用测试账户登录:
- 用户名: `swaggertest`
- 密码: `test123456`

### 4️⃣ 开始对话 (2分钟)

1. 点击 "新建对话" 创建对话
2. 在输入框输入消息
3. 按 Enter 发送
4. 查看 AI 回复

## 常见操作

### 发送消息
- 输入文本后按 **Enter** 发送
- **Shift+Enter** 换行

### 上传文件
1. 点击输入框左边的 **➕** 按钮
2. 选择文件
3. 文件信息会附在消息中

### 创建新对话
1. 点击左侧 "新建对话" 按钮
2. 开始新的对话

### 切换对话
1. 点击左侧对话列表的任一项
2. 历史消息会加载显示

## 常见问题

**Q: 无法连接到后端?**
- 确保后端服务器运行: `uv run uvicorn app.main:app --reload`
- 检查后端地址是否正确
- 检查浏览器控制台的错误信息

**Q: 登录失败?**
- 确认用户名和密码正确
- 检查后端是否运行
- 清除浏览器缓存后重试

**Q: 消息发送后没有响应?**
- 检查网络连接
- 查看浏览器开发者工具的 Network 标签
- 检查后端 DeepSeek API 配置

**Q: 界面显示不正常?**
- 刷新页面
- 清除浏览器缓存
- 检查浏览器版本是否最新
